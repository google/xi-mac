## General

| Action            | Shortcut      |
| :---------------- | :------------ |
| Open a new window | Command (⌘)-N |
| Open a new tab    | Command (⌘)-T |
| Open preferences  | Command (⌘)-, |
| Quit application  | Command (⌘)-Q |

## Find and replace

| Action           | Shortcut      |
| :--------------- | :------------ |
| Open find dialog | Command (⌘)-F |
